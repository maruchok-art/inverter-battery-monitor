# 🔋 Моніторинг резервного живлення будинку

Автоматизована система моніторингу стану акумуляторів (інвертор Deye) для багатоквартирного будинку. Інвертор забезпечує роботу критичної інфраструктури (водяні насоси, ліфт тощо) під час відключень електроенергії. Скрипт регулярно перевіряє залишок заряду та публікує зручні сповіщення у Telegram-чат мешканців.

**Особливий акцент у системі зроблено на безпеці ліфта:** бот попереджає про низький заряд батареї, щоб запобігти застряганню людей у кабіні під час повного розряду системи.

## ✨ Головні функції

* **Моніторинг Deye/Solarman API:** Автоматично зчитує поточний рівень заряду загального акумулятора (SOC - State of Charge) інвертора Deye.
* **Розумні сповіщення в Telegram:** Надсилає попередження у будь-який налаштований інформаційний чат будинку (наприклад, "СвітлоБот"). Пороги сповіщень:
  * 🟡 **< 95%:** Будинок перейшов на акумулятори (тихе сповіщення).
  * 🟠 **< 50%:** Запас енергії обмежений. Прохання максимально скоротити використання ліфта та йти сходами.
  * 🔴 **< 30%:** Критичний заряд! Заборона користування ліфтом (високий ризик зупинки кабіни між поверхами).
  * ⚠️ **OFFLINE:** Втрачено зв'язок з інвертором (зник інтернет або живлення роутера системи).
  * ✅ **Відновлення зв'язку:** Сповіщає про повернення інвертора в мережу з вказівкою поточного заряду системи.
* **Кінцева машина станів (State Machine):** Скрипт запам'ятовує свій поточний стан, щоб **не дублювати** повідомлення. Під час заряджання батареї перехід між станами відбувається "мовчки", без зайвого спаму в чат.
* **Веб-дашборд:** Гарний та адаптивний `dashboard.html` для перегляду поточного заряду та статусу системи в реальному часі прямо з браузера чи смартфона.
* **Оптимізація API (Анти-бан):** Використовує базу даних JSONBlob для кешування токена доступу (діє 12 годин), щоб уникнути блокування від серверів Solarman за занадто часті запити авторизації.

## 🏗 Архітектура проєкту

1. **`main.py` (Python):** Ядро системи. Запускається за розкладом через GitHub Actions.
2. **JSONBlob:** Безкоштовне хмарне сховище, яке діє як "пам'ять" системи. Зберігає токен авторизації, поточний рівень тривоги, час останнього оновлення та % заряду.
3. **`dashboard.html`:** Фронтенд-частина. Зчитує дані напряму з JSONBlob (через `corsproxy.io`) та візуалізує їх у вигляді зручного віджета.
4. **Telegram Bot API:** Відповідає за доставку повідомлень у цільовий чат (налаштовується через ID чату).

## 🚀 Встановлення та налаштування

### 1. Підготовка бази даних (JSONBlob)
1. Перейдіть на [jsonblob.com](https://jsonblob.com/) і створіть новий порожній об'єкт: `{}`.
2. Збережіть його і скопіюйте унікальний ID з URL (наприклад, `019d1fde-f71e-7530-83fb-a58fe38ac1c7`).

### 2. Налаштування GitHub Secrets
У вашому репозиторії перейдіть до **Settings** -> **Secrets and variables** -> **Actions** і додайте наступні змінні (`New repository secret`):

* `TG_BOT_TOKEN`: Токен вашого Telegram бота (від @BotFather).
* `TG_CHAT_ID`: ID групи або каналу будинку, куди будуть публікуватися сповіщення.
* `SOLARMAN_APP_ID`: Ваш App ID від Solarman Developer Portal.
* `SOLARMAN_APP_SECRET`: Ваш App Secret від Solarman.
* `SOLARMAN_EMAIL`: Email від акаунта Solarman.
* `SOLARMAN_PASSWORD`: Пароль від акаунта Solarman.
* `DEVICE_SN`: Серійний номер інвертора / логера Deye.
* `JSONBLOB_ID`: ID вашої бази з кроку 1.

### 3. Налаштування GitHub Actions
Створіть файл `.github/workflows/main.yml` для автоматичного запуску скрипта кожні 10 хвилин:

```yaml
name: Run battery check script

on:
  schedule:
    - cron: '*/10 * * * *' # Запуск кожні 10 хвилин
  workflow_dispatch: # Дозволяє запускати скрипт вручну

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.10'
      - name: Install dependencies
        run: pip install requests
      - name: Run script
        env:
          TG_BOT_TOKEN: ${{ secrets.TG_BOT_TOKEN }}
          TG_CHAT_ID: ${{ secrets.TG_CHAT_ID }}
          SOLARMAN_APP_ID: ${{ secrets.SOLARMAN_APP_ID }}
          SOLARMAN_APP_SECRET: ${{ secrets.SOLARMAN_APP_SECRET }}
          SOLARMAN_EMAIL: ${{ secrets.SOLARMAN_EMAIL }}
          SOLARMAN_PASSWORD: ${{ secrets.SOLARMAN_PASSWORD }}
          DEVICE_SN: ${{ secrets.DEVICE_SN }}
          JSONBLOB_ID: ${{ secrets.JSONBLOB_ID }}
        run: python main.py

### 4. Веб-дашборд
Файл `dashboard.html` можна розмістити на будь-якому безкоштовному хостингу (наприклад, **GitHub Pages**). 
*Обов'язково переконайтеся, що змінна `JSONBLOB_ID` всередині тегу `<script>` у файлі `dashboard.html` збігається з вашим ID бази.*
